# Cellular Automaton
- USING p5 framework


# Wolfram's Elementary Cellular Automaton

In the /wolfram folder, you can run to see how a visual representation of cellular automaton rule which starts with rule 250. You can edit the ruleset variable to visualize a different rule. You can also start off with a random array to see a more random example. 

# Conway's Game of Life

In the /game-of-life folder, you can run John Conway's Game of Life, which starts off with a random array. Run it to see the game of life!

# To Run

To easily run it in your browser, I used the Live Server extension in VS Code.
